# plala-maintenance-check

plalaのメンテナンス・故障情報をチェックするためのツール



## Usage

```bash
# 対象地域を引数に入力して実行
yarn run dev {area}

# 例) 東京都のメンテナンス・故障情報をチェック
yarn run dev tokyo
```
